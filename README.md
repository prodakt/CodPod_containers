# CodPod_containers

https://hub.docker.com/search?q=CPC2
https://bio.tools/biocontainers

W tego typu repozytoriach, ale też na oficjalnych stronach odszukać linki do kontenerów programó do przewidywania potencjału kodowania. 
9-11 programów wystarczy, np.:
- CNCI
- CPC2
- lgc
- PLEK
- CPAT
- CPPred
- longdist
- PredLnc-GFStack
- LncADeep
- FEELnc
- LncFinder
- ... lub inne, 9 to minimum, ale im wiecej tym lepiej.

Pobrać kontenery, przetestować czy działają, opisać procedury odpalania oraz krótko scharakteryzować algorytmy. 
Najwazniejsze, zeby zbudować (z użyciem przygotowanych danych testowych - nsze sekwencje do testowania, trenowania i przewidywania) zestaw procedur dla każdego programu oddzialnie:
- pobieranie dockera,
- odpalanie analizy,
- ekstrakcja wyników.
